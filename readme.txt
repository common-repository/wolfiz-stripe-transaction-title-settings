Stripe Payment Transaction Title Settings With Woocommerce adds a tab in woocommerce settings to change how the transaction title appears in Stripe transactions list.

== Description ==

Wolfiz Stripe Payment title Settings plugin adds a new tab in woocommerce settings. It gives three settings of checkboxes.

1) Title of the product.
2) Order name.
3) Quantity.

Enable these settings if you want to alter Title shown in dashboard of stripe payment menu. 

Enabling Title of the product setting will show product name in the title of stripe payment menu.

Enabling Order name setting will show woocommerce Order name in the title of stripe payment menu.

Enabling Quantity setting will show woocommerce item quantity in the title of stripe payment menu.


== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress.

3. Use the Woocommerce > Settings > Custom Stripe Settings to configure the custom stripe settings.

== FAQ == 

Manual Installation

Navigate to the 'Add New' in the plugins dashboard.

Navigate to the 'Upload' area.

Select wolfiz_stripe_settings.zip from your computer.

Click 'Install Now'.

Activate the plugin in the Plugin dashboard.


Using FTP

Download wolfiz_stripe_settings.zip

Extract the wolfiz_stripe_settings.zip directory to your computer

Upload the wolfiz_stripe_settings.zip directory to the /wp-content/plugins/ directory

Activate the plugin in the Plugin dashboard

Q) Is there any effect on title of woocommerce orders by using this plugin

A) No, this plugin only modify title of transaction forwarded to stripe.

Q) Will this plugin work with any other Form/plugin where stripe is enabled as payment gateway.

A) No, this plugin will only work with woocommerce.
